Raw data tab

In the raw data tab you can select data from the table that you want to plot.
After you have selected the desired data, press the filter data button to get rid of the other data and plot the selected data in the plot tab.

Plotting tab

In the plotting tab you can see the plotted version of the data.
You can choose to plot the raw data and the data after processing using the checkboxes.
  • The top plot is the time domain version of the data.
  • The middle plot is the magnitude component of the FFT of the data.
  • The bottom plot is the phase component of the FFT of the data.

Settings tab

In the settings tab you can select all possible settings.

Window Functions

The window functions are used to avoid spectral leak.
For more info on each window function click here.

Smoothing

Smoothing algorithms can be used to smooth out the high frequencies in the signal. If you only want these frequencies you can use the math settings.
For more info on each smoothing algorithm click here.

Math Settings

The math settings can be used to perform certain math calculations between the raw data and the processed data.
You can select a sign for the raw data. For the processed data you can choose to substract, add, divide, multiply with the raw data.
